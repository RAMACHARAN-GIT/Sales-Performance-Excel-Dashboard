
# Sales Performance Dashboard (Excel)

This project presents an **interactive Sales Performance Dashboard built in Microsoft Excel** to analyze business sales data. The dashboard provides insights into sales performance across different categories, regions, and customer segments.

---

## Project Objective

The goal of this project is to transform raw sales data into meaningful insights using Excel data analysis and visualization tools.

This dashboard helps answer questions such as:

- What is the total sales and profit?
- Which region generates the most revenue?
- Which product categories perform best?
- What are the top-performing subcategories?
- How do different customer segments contribute to sales?

---

## Dashboard Features

### KPI Cards
- Total Sales
- Total Profit
- Total Quantity Sold
- Average Discount

### Charts & Visualizations
- Sales by Category
- Sales by Region
- Sales by Segment
- Top 10 Subcategories by Sales

### Filters (Slicers)
- Region
- Segment
- Category
- Ship Mode

These slicers allow users to dynamically filter and explore the data.

## Data Analysis Process

1. Data Cleaning
   - Removed duplicates
   - Checked missing values
   - Standardized column formats

2. Data Preparation
   - Structured raw dataset
   - Created Pivot Tables for analysis

3. Visualization
   - Built charts from pivot tables
   - Designed KPI cards

4. Dashboard Creation
   - Combined charts and KPIs into a single interactive dashboard
   - Added slicers for dynamic filtering

## Key Insights

- Sales performance varies significantly across regions.
- Certain product categories contribute the majority of revenue.
- A small number of subcategories generate a large portion of total sales.
- Customer segments show different purchasing patterns.
